centos最小化安装常用软件

发表于2021-02-20|更新于2021-02-20|Code
|字数总计:205|阅读时长:1分钟|阅读量:

改网卡自启动

  • 修改网卡配置文件，将ONBOOT的值改为yes

vi /etc/sysconfig/network-scripts/ifcfg-ens32

  • 重启网络服务

service network start

改yum源

  • 先装个wget

yum -y install wget

  • 备份yum仓库文件

mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

  • 阿里云仓库手册

https://developer.aliyun.com/mirror/centos?spm=a2c6h.13651102.0.0.53322f70Adftrz

  • 替换成阿里云仓库

wget -O /etc/yum.repos.d/CentOS-Base.repo https://mirrors.aliyun.com/repo/Centos-7.repo

  • 清除缓存

yum clean all

  • 新建缓存

yum makecache

  • 如果新建缓存失败可能是dns解析错误，手动配置dns为8.8.8.8即可

安装常用软件

net-tools

yum -y install net-tools

wget、vim、mlocate、lsof、rz、zip

yum -y install wget vim mlocate lsof lrzsz unzip zip

gcc全家桶

yum -y install gcc*

openssl-devel

yum -y install openssl-devel

bash-completion

yum -y install bash-completion
